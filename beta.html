As A User

Ready to send some private, on-chain transactions?

When Moonlight reaches production, you'll use it from the UI of your financial institution. It will feel like using any other crypto wallet or bank's "send/receive" functionality. Until then, it requires some manual setup!

Start by installing our Chrome Browser Extension.

After it's installed, you'll need to:

  1. Create/import account
  2. Add a privacy channel (a contract ID)
  3. Add a privacy provider (a URL)

Here's a little more info about each step:

1. Create/import account

If you're familiar with setting up other crypto wallets, this should be straightforward. Follow the steps to create a new wallet account. Make sure you are using the Testnet network.

2. Add a privacy channel

For the current beta test, the Moonlight team has deployed a single privacy channel on Stellar's Testnet. We'll explain more about this soon; for now all you need to know is that each privacy channel is a smart contract. You need to know the privacy channel's contract ID, and you can optionally give it a name (this name is just for you; you can set it to whatever you want).

In the Moonlight Wallet extension, find the "Private Channels" / "Moonlight Channels" list. It will be empty, for now. Click the `+` to configure a new one. Set "Network" to "Test". Set "Contract ID" to "C…". Set the name to whatever you want. "Moonlight Beta" is reasonable.

3. Add a privacy provider

This is a financial institution. In the production version of Moonlight, you would access Moonlight via their UI. For now, you need to know their URL. (If you are also testing Moonlight as a privacy provider, you will have already configured/provisioned your service following the "As A Bank" instructions. You can now paste in your URL.)

The Moonlight extension allows you to specify a "Provider Name"—this is your own private note. Set it to anything you want.
